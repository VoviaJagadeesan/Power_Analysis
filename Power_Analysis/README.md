# PJM Energy Consumption Forecast Dashboard

This project predicts hourly electricity demand using an LSTM model.

Dataset:
PJM Interconnection Hourly Energy Consumption.

Features
- Energy demand trend visualization
- Hourly consumption pattern analysis
- Seasonal demand analysis
- LSTM based forecasting
- 30-day energy demand prediction
- Forecast insights and operational recommendations

Run the application

streamlit run app.py
